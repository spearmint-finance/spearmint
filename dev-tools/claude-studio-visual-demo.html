POST /api/scenarios/preview
Simulate a financial scenario without saving it to the database. Returns baseline projection vs scenario projection with KPIs and deltas for real-time scenario exploration.
📥 Request Body (application/json)
name string optional // Scenario name for reference
horizonMonths integer required // Forecast period (1-60 months)
startingBalance decimal required // Initial account balance
adjusters array<Adjuster> required // Scenario modifications
type string required // job_loss | income_reduction | expense_change
targetPersonId integer optional // Person affected by adjustment
params object required // Type-specific parameters
startDate date optional // When adjustment begins
endDate date optional // When adjustment ends (null = permanent)
📤 Response (200 OK) (application/json)
baselineSeries array<TimeSeriesPoint> required // Original projection
date date // Projection date
income decimal // Projected income
expenses decimal // Projected expenses
netCashFlow decimal // Net cash flow
balance decimal // Ending balance
scenarioSeries array<TimeSeriesPoint> required // Modified projection with adjusters
kpis object required // Key performance indicators
runwayMonths decimal // Months until funds depleted
minBalance decimal // Minimum balance in period
deltas object required // Changes from baseline
income decimal // Total income change
expenses decimal // Total expenses change
netCashFlow decimal // Net cash flow change
generatedAt datetime required // Timestamp of generation
⚠️ Error Responses
400 // Invalid scenario configuration
500 // Server error during projection calculation
