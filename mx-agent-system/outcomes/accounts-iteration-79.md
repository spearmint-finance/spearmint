# Accounts Iteration 79 — Account Active/Inactive Toggle

**Date:** 2026-03-22
**PR:** #220
**Focus:** Add active/inactive toggle to account details dialog

## Before
- Users could deactivate accounts but had no way to reactivate them from the UI
- No visual indicator for inactive accounts in the details dialog

## After
- Active/Inactive switch in the account edit form
- Warning "Inactive" chip shown in dialog title for inactive accounts
- Status displayed in read-only details view

## Outcome
- Human intervention: No
- Measurable: Yes — inactive accounts can now be reactivated
- Regressions: None
